Análisis Predictivo

Este proyecto se centra en la construcción de modelos para realizar predicciones basadas en datos históricos. Se exploraron patrones clave en los datos y se aplicaron técnicas de machine learning para generar predicciones precisas.

  • Técnicas utilizadas: Regresión lineal, Árboles de decisión.
  • Herramientas: Python, Scikit-learn.
