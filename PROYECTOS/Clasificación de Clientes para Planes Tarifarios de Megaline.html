Clasificación de Clientes para Planes Tarifarios de Megaline

Resumen del Proyecto

Megaline, una compañía de telefonía móvil, busca optimizar la migración de sus clientes hacia nuevos planes tarifarios. Actualmente, muchos usuarios continúan utilizando planes heredados, lo que representa un desafío en la adopción de los planes Smart y Ultra.

Este proyecto tiene como objetivo desarrollar un modelo de clasificación que, basado en el comportamiento de los clientes, recomiende el plan más adecuado.

El modelo se evaluará utilizando la métrica de exactitud (accuracy) y debe alcanzar un mínimo de 0.75 para ser considerado satisfactorio.

Metodología Utilizada

1. Exploración y Análisis de Datos (EDA)

  • Carga del dataset con información del uso de servicios por los clientes.
  • Análisis de la distribución de variables numéricas y categóricas.
  • Identificación de valores nulos y estrategia de imputación.
  • Análisis de correlación entre variables y la variable objetivo (Plan: Smart o Ultra).

2. Preprocesamiento de Datos

  • Conversión de datos categóricos mediante One-Hot Encoding o Label Encoding.
  • Normalización de variables numéricas para mejorar la eficiencia del modelo.
  • Separación del dataset en conjuntos de entrenamiento y prueba.

3. Entrenamiento de Modelos de Clasificación

  • Entrenamiento de modelos como Regresión Logística, Árboles de Decisión y Random Forest.
  • Optimización de hiperparámetros para mejorar el rendimiento.

4. Evaluación del Modelo

  • Evaluación de los modelos utilizando la métrica de exactitud (accuracy).
  • Selección del modelo con mejor rendimiento para recomendaciones futuras.

Resultados

El modelo de Random Forest alcanzó una exactitud del 78%, superando el umbral establecido de 75%. Este modelo demostró ser efectivo en la clasificación de clientes para recomendarles el plan más adecuado.

Conclusiones

La implementación de modelos de clasificación permite a Megaline identificar y recomendar planes tarifarios óptimos para sus clientes, facilitando la migración desde planes heredados hacia opciones más modernas y beneficiosas tanto para la empresa como para los usuarios.
