Análisis de Series Temporales para Pronóstico de Ventas

Este proyecto utiliza modelos de series temporales para predecir ventas futuras. Se analizaron patrones históricos de datos de ventas para desarrollar un modelo que facilite la toma de decisiones estratégicas.

  • Técnicas utilizadas: Análisis de series temporales, Modelos ARIMA.
  • Herramientas: Python, Pandas, Statsmodels.
