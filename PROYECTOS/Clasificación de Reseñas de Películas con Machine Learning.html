Clasificación de Reseñas de Películas con Machine Learning

Resumen del Proyecto

Este proyecto tiene como objetivo clasificar automáticamente reseñas de películas en positivas o negativas utilizando modelos de machine learning y procesamiento de lenguaje natural (NLP). Se entrenaron distintos modelos para analizar el sentimiento de los textos y determinar la mejor estrategia para predecir correctamente si una reseña es favorable o desfavorable. El principal criterio de evaluación es la métrica F1-score, con un objetivo mínimo de 0.85.

Metodología Utilizada

1. Exploración y Preparación de Datos

  • Carga del dataset de reseñas de películas de IMDb.
  • Revisión de estructura del dataset y distribución de clases.
  • Análisis de valores nulos y eliminación de datos innecesarios.
  • Análisis de la cantidad de palabras en cada reseña y su impacto en la clasificación.

2. Preprocesamiento de Texto

Para transformar los textos en datos utilizables por los modelos de machine learning, se aplicaron técnicas de Procesamiento de Lenguaje Natural (NLP), tales como:

  • Conversión del texto a minúsculas.
  • Eliminación de caracteres especiales y puntuación.
  • Tokenización de las palabras.
  • Eliminación de palabras vacías (stopwords).
  • Lematización para reducir las palabras a su forma raíz.

3. Vectorización

Se utilizaron técnicas de vectorización para convertir el texto preprocesado en representaciones numéricas:

  • Bag of Words (BoW).
  • TF-IDF (Term Frequency-Inverse Document Frequency).

4. Entrenamiento de Modelos

Se entrenaron varios modelos de clasificación, incluyendo:

  • Regresión Logística.
  • Máquinas de Soporte Vectorial (SVM).
  • Árboles de Decisión.
  • Random Forest.
  • Redes Neuronales.

5. Evaluación de Modelos

Los modelos se evaluaron utilizando las siguientes métricas:

  • Precisión.
  • Recall.
  • F1-score.
  • Matriz de confusión.

Resultados

El modelo de Regresión Logística con vectorización TF-IDF alcanzó un F1-score de 0.87, superando el objetivo establecido de 0.85. Este modelo demostró ser efectivo en la clasificación de reseñas de películas en positivas y negativas.

Conclusiones

La combinación de técnicas de preprocesamiento de texto y modelos de machine learning permite clasificar eficazmente las reseñas de películas según su sentimiento. El modelo desarrollado puede ser utilizado para analizar grandes volúmenes de reseñas y proporcionar insights valiosos sobre la percepción de las películas por parte del público.
