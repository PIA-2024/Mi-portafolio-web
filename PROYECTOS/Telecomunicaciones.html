Predicción de la Tasa de Cancelación de Clientes

Detalles del Proyecto

Este proyecto tiene como objetivo predecir la tasa de cancelación de clientes en el sector de telecomunicaciones. Para ello, se analizaron datos históricos de contratos, uso de servicios y características demográficas. Utilizando técnicas de machine learning, se entrenaron modelos que permiten identificar patrones de comportamiento asociados a la cancelación de servicios.

Ver en GitHub

Metodología

  • Datos: Contratos, datos personales, servicios de internet y servicios telefónicos.
  • Técnicas: Ingeniería de características, modelos de clasificación supervisada.
  • Modelos evaluados: Random Forest, LightGBM, Regresión Logística.
  • Herramientas: Python, Scikit-learn, LightGBM, Pandas, Matplotlib.
  • Métrica principal: AUC-ROC (mejor modelo: LightGBM con 0.92).

Conclusiones

El modelo desarrollado con LightGBM permitió identificar los principales factores asociados a la cancelación de clientes, lo que brinda a la empresa herramientas clave para implementar estrategias de retención. Las características más relevantes incluyeron el tipo de contrato, antigüedad del cliente y uso de servicios adicionales. Este proyecto demuestra el valor del análisis predictivo en la toma de decisiones estratégicas.
