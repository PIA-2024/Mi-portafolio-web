Predicción de Abandono de Clientes en un Banco

Descripción del Proyecto

El objetivo de este proyecto es **predecir si un cliente abandonará el banco en el corto plazo** utilizando modelos de Machine Learning. Se exploraron métodos de clasificación supervisada para optimizar la tasa de retención y reducir la pérdida de clientes.

Objetivos

  • Desarrollar un modelo de predicción de abandono de clientes.
  • Analizar patrones y factores que influyen en la deserción.
  • Proporcionar insights para mejorar la retención de clientes.

Metodología

Se realizó un análisis exploratorio de datos (EDA) para entender las variables clave. Luego, se implementaron técnicas de preprocesamiento como normalización y codificación de variables categóricas. Se probaron múltiples modelos de clasificación incluyendo **Regresión Logística, Árboles de Decisión, Random Forest y XGBoost**. Finalmente, se evaluó el rendimiento con métricas como **AUC-ROC, F1-score y Matriz de Confusión**.

Resultados

El modelo de **Random Forest** obtuvo el mejor rendimiento con un **F1-score de 0.87** y un **AUC-ROC de 0.90**, demostrando alta precisión en la predicción del abandono de clientes.

Conclusiones

Los factores más influyentes en la predicción del abandono fueron:

  • Saldo de la cuenta.
  • Tiempo de permanencia en el banco.
  • Uso de productos financieros.
Se recomienda implementar estrategias de fidelización personalizadas para retener a los clientes en riesgo de abandono.
